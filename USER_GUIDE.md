## Troubleshooting

### I Forgot My Password

**For Customers:**
1. Click **"Forgot Password?"** on login page
2. Enter your email address
3. Check your email for reset link
4. Click link and create new password
5. Login with new password

**For Staff/Admin:**
1. Click **"Forgot Password?"** on login page
2. Enter your email address
3. Check email for reset link
4. Set new password

### My Order Isn't Showing

**Possible causes:**
1. You're logged into wrong account - verify email
2. Order was placed via WhatsApp - check with admin
3. Browser cache issue - try incognito/private mode
4. Order from old account - login to that account

**Solution:**
1. Logout completely
2. Clear browser cache (Ctrl+Shift+Delete)
3. Login again
4. Check "My Orders" page

### I Haven't Received a Notification

**Possible causes:**
1. Browser notifications are disabled
2. Pop-ups are blocked
3. Network connection issue

**Solution:**
1. Check browser notification settings
2. Allow notifications from website
3. Refresh the page
4. Manually check "My Orders" page for updates

### Staff Member Can't Login

**As Admin:**
1. Go to Staff Management
2. Verify staff member email
3. Click "Reset Password"
4. Staff will receive reset email
5. They create new password and login

### Duplicate Order Was Placed

**As Admin:**
1. Go to "All Orders"
2. Find the duplicate order
3. Click order and select "Cancel Order"
4. Mark as "Duplicate - Canceling per admin request"
5. Optional: Create credit note for customer

**As Customer:**
1. Contact admin immediately
2. Provide both order IDs
3. Admin will cancel the duplicate
4. Refund processed (if applicable)

### Order Status Not Updating

**Possible causes:**
1. Staff hasn't updated status yet
2. Network connectivity issue
3. Browser cache problem

**Solution:**
1. Refresh the page (F5)
2. Clear browser cache
3. Contact staff if in transit
4. Contact admin if issue persists

### I'm Receiving Too Many Notifications

**For Customers:**
1. Go to **Settings** (profile menu)
2. Click **"üîî Notification Preferences"**
3. Uncheck notifications you don't want
4. Save preferences

**For Admin:**
1. Go to **Settings**
2. Adjust notification settings
3. Save changes

## Frequently Asked Questions (FAQ)

**Q: How long does delivery take?**
A: Typical delivery time is 30-60 minutes depending on location and current orders.

**Q: What if I'm not home when the staff member arrives?**
A: Staff will call you before delivery. If you're not available, contact us to reschedule.

**Q: Can I cancel my order?**
A: Contact admin immediately. Orders can be cancelled if not yet assigned.

**Q: Do you accept card payments?**
A: Currently only Cash on Delivery is available.

**Q: Can I modify my order after placing it?**
A: Contact admin immediately. Modifications depend on preparation stage.

**Q: Are there group discounts?**
A: Contact sales team for bulk orders and special pricing.

**Q: How do I become a staff member?**
A: Interested staff should apply to admin. Admin creates accounts for qualified candidates.

**Q: How is my personal data protected?**
A: Your data is encrypted and used only for order processing and delivery.

**Q: What if I have a problem with my order?**
A: Contact support immediately with your order ID. We'll investigate and resolve quickly.

**Q: How can I track multiple orders?**
A: Login to your account and go to "My Orders" to see all your orders with tracking.
